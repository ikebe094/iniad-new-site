コンソール・クイズの解答例

INIADの1年次の必修科目「コンピュータサイエンス概論」ではPythonというプログラミング言語を学習します。興味がある方は調べてみましょう。

模範解答


            def max_len(xs):
                maxval = -1
                for x in xs:
                    if len(x) > maxval:
                        maxval = len(x)
                return maxval

ダメな解答例1


            def max_len(xs):
                maxval = -1
                for x in xs:
                   if len(x) > maxval:
                       maxval = len(x)
            print(maxval)  # この行が違う

「関数を定義してください」という問題なので、結果をprintすると答えを返すことができません。

ダメな解答例2


            def max_len(xs):
                maxval = -1
                for x in xs:
                    if len(x) > maxval:
                        maxval = len(x)
                return maxval
            max_len(input())  # この行が余計

「関数を定義してください」とだけ書いてあるので、関数定義ができていれば他は不要です。

なお、この問題は定期試験で実際に出題された問題です。プログラムの記述式問題では、エラーなくプログラムが実行できること、プログラムの動作をもとに正解・不正解を判断します。余分なコードや、大文字小文字の区別、スペルミスなどに十分注意しましょう。動作例がついている場合は、これを使って動作確認をすることも重要です。
