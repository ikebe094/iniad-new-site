INIAD API

APIを使えるようになろう

INIADの学生には、一人一つのロッカーが割り当てられています。学生はこれをICカードやスマートフォンを使用して開けることができます。優秀な学生はすでに様々なアイデアを使用して便利なアプリケーションを作成しています。

増設可能な設備

INIAD学内には天井が直接操作可能になっています。これは、学生や教職員の手で改造できるようにするためのものです。学生には一部のAPIのみが使用できますが、優秀な学生には特別に他のAPIを使用できるかもしれません。
